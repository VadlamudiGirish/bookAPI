# Node.js + TypeScript REST API

A simple REST API for managing books, built with Node.js, Express, and TypeScript. This project demonstrates CRUD operations with in-memory storage.

## Features

- CRUD operations for book resources
- TypeScript for type safety
- Express.js for routing
- Nodemon for development reloading
- Built-in API documentation

## Prerequisites

- Node.js (v14+)
- npm (comes with Node.js)
